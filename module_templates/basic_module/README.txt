Basic module template

What to edit first:
- module id
- module slug
- titles and descriptions
- topic ids and slugs
- lesson texts
- tasks and answers

You can import this folder from the app settings,
or copy it into the main modules folder after editing.

Important:
- ids must stay unique across all modules
- slug should also be unique
- answers can be a string or a list
- options are used for choice questions
