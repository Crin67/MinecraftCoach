# Module Templates

This folder contains starter templates for new learning modules.

Recommended workflow:

1. Copy `basic_module/` to a new working folder.
2. Edit `module.json`.
3. Change every stable identifier:
   - `id`
   - `slug`
   - topic ids
   - task ids
4. Import the folder from the app settings, or copy the finished folder into `modules/`.

The app now supports:

- module folder import
- `.zip` import
- `module.json` import
- direct loading from `modules/<name>/module.json`
