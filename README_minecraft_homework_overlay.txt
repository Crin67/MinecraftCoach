Программа: Minecraft Homework Overlay

Что делает:
- Раз в 5-10 минут показывает ребёнку задание поверх экрана.
- Пока ответ не решён, окно остаётся сверху.
- Есть задания по математике, вставке букв и чтению.
- Подходит как базовый прототип для детей 5-8 лет.

Важно:
- Лучше запускать Minecraft в оконном или borderless режиме.
- Программа ПЫТАЕТСЯ нажать ESC, если активное окно называется Minecraft.
- Это помогает поставить игру на паузу, но 100% гарантию для всех версий/модов дать нельзя.
- Для заданий на чтение сейчас нужна кнопка взрослого: "Прочитано!"

Как запустить на Windows:
1. Установите Python 3.11+.
2. Сохраните файл minecraft_homework_overlay.py.
3. Запустите двойным кликом или через команду:
   python minecraft_homework_overlay.py

Как собрать в .exe:
1. Откройте cmd в папке с файлом.
2. Установите PyInstaller:
   pip install pyinstaller
3. Соберите:
   pyinstaller --onefile --windowed minecraft_homework_overlay.py
4. Готовый exe будет в папке dist.
